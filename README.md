Задание:
Вариант 4. 0-1 Рюкзак с жадным алгоритмом 
Задача: реализовать жадный приближенный алгоритм для 0-1 рюкзака. Отличие от дробного 
в том, что предметы нельзя дробить. 
Требования: - Реализовать 2-аппроксимационный жадный алгоритм - Вывести список выбранных предметов и общую стоимость - Указать коэффициент аппроксимации 
Входные данные: 
Предметы: {(вес=3, стоимость=12), (вес=2, стоимость=8), 
(вес=4, стоимость=20), (вес=5, стоимость=18)} 
Вместимость: 7

Вывод кода:
Выбранные предметы:
Вес=4, Стоимость=20
Вес=3, Стоимость=12
Общая стоимость: 32
Коэффициент аппроксимации: 1.00

Описание алгоритма:
1. Сортировка по отношению стоимости к весу (value/weight) для выбора наиболее выгодных предметов.
2. Пошаговое добавление weight + totalweight, пока не превышено ограничение по весу(c).
3. Выбор полного заполнения — это жадный подбор по стоимости для получения верхней границы стоимости.
4. Коэффициент аппроксимации — отношение стоимости выбранных предметов(totalvalue) к стоимости полного заполнения(fullvalue), что показывает качество приближения.

Временная сложность: O(n log n)
Почему она такая: Основная операция — сортировка списка предметов по отношению стоимости к весу, которая занимает O(n log n). После сортировки, цикл по предметам в худшем случае проходит O(n) раз, так как рассматривает каждый предмет один раз. Внутри цикла выполняются простые операции сравнения и присваивания — O(1). Временная сложность в сумме равна: O(n log n) + O(n) = O(n log n).

Ответ на вопрос:

Алгоритм:

Пока есть непокрытые ребра:
1. Берем любое ребро (u,v)
2. Добавляем u и v в покрытие
3. Удаляем все ребра, инцидентные u или v

Доказательство 2-аппроксимации:

1. Паросочетание: Выбранные алгоритмом ребра не пересекаются (образуют паросочетание A)
2. Нижняя граница для OPT: Любое вершинное покрытие (включая оптимальное OPT) должно содержать больше одной вершины из каждого ребра паросочетания A
⇒ |OPT| ≥ |A|
3. Размер нашего решения: Алгоритм добавляет 2 вершины на каждое ребро из A
⇒ |C| = 2 × |A|
4. Сравнение:
|C| = 2 × |A| ≤ 2 × |OPT|

Значит, покрытие C не более чем в 2 раза хуже оптимального.
